Project Proposal

I would like to work on the marketplace project. I'd like it to be a thrifted marketplace where people can buy and sell used items they do not need anymore. I'd also like to implement a 'swap' option with which you can exchange items of similar value with other users. The marketplace would be divided into different categories: clothing, home decor, appliances and electronics and each one would have several subcategories with its corresponding features (e.g an item listed in the clothing/shoes section would have attributes such as shoe size, style(sandal, boot, etc.), brand, condition(like new, slightly used,etc.)).
